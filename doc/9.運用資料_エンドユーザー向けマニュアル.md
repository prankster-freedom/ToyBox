# エンドユーザー向け説明書 (USER_MANUAL.md)

ToyBox アプリケーションの利用方法について説明します。

## 1. アプリケーションへのアクセス

ブラウザでアプリケーションの URL にアクセスします。
ローカル環境の場合は `http://localhost:3000` です。

## 2. 機能一覧

### 2.1. ログイン

アプリケーションの一部の機能（チャット履歴の保存など）を利用するには、Google アカウントでのログインが必要です。

1.  画面右上の「Login with Google」ボタン（または同様のリンク）をクリックします。
2.  Google の認証画面に遷移するので、アカウントを選択してログインします。
3.  認証が完了するとトップページに戻り、ログイン状態となります。

### 2.2. ログアウト

1.  ログイン状態で、画面に表示されている「Logout」ボタンをクリックします。
2.  ログアウトが完了し、トップページ（未ログイン状態）に戻ります。

### 2.3. 挨拶機能 (Greet)

シンプルな API テスト機能です。

1.  画面上の入力フィールドに名前を入力します。
2.  「Greet」ボタンをクリックします。
3.  サーバーから「Hello, {名前}!」というメッセージが返され、画面に表示されます。

### 2.4. チャット機能 (AI Chat)

AI エージェントと会話を楽しむことができます。

1.  チャット画面に移動します（またはトップページに統合されている場合）。
2.  メッセージを入力し、送信します。
3.  AI からの応答が表示されます。
4.  会話を続けることで、AI はあなたの性格や好みを学習し、応答を変化させる場合があります（Daydream/Dream 機能）。

## 3. トラブルシューティング

- **画面が表示されない**: サーバーが起動しているか確認してください。
- **エラーが表示される**: ブラウザのコンソールログを確認するか、管理者に問い合わせてください。
